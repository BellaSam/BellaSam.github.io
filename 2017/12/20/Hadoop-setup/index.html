JAVA Installation

  1. sudo apt-get install software-properties-common
  2. sudo add-apt-repository ppa:webupd8team/java
  3. sudo apt-get update
  4. sudo apt-get install oracle-java8-installer

  5. vi /etc/profile, add JAVA_HOME, JRE_HOME:

  6. source /etc/profile

Hadoop Installation

  1. wget https://archive.apache.org/dist/hadoop/common/hadoop-1.2.1/hadoop-1.2.1.tar.gz
  2. mv hadoop-1.2.1.tar.gz /opt/
  3. cd /opt
  4. tar -zxvf hadoop-1.2.1.tar.gz

  5. vi /opt/conf/hadoop-env.sh, add JAVA_HOME

  6. vi /opt/conf/core-site.xml

  7. vi /opt/conf/hdfs-site.xml

  8. vi /opt/conf/mapred-site.xml

  9. vi /etc/profile, add HADOOP_HOME

  10. source /etc/profile

  11. hadoop namenode -format
  12. cd /opt/hadoop-1.2.1/bin, run start-all.sh

  13. jps, check java thread

  14. check file system : hadoop fs -ls /
